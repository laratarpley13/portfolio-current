About Me

Projects

CookIt

Live App Client Repo API Repo

I love to cook and create new recipes and wanted a platform that would make it easy for anyone to share their recipe creations and recieve constructive feedback on the recipes in the form of pictures, as I feel like food can be a very visual experience.

This app allows users to create their own account and explore recipes, with the option to filter by category or dietary restrictions, and try other people's recipes and leave them feedback as well.

Technology Client: React, ReactRouter, HTML, CSS

Techonology API: Node.js, Express, PostgreSQL

List Manager

Live App Client Repo API Repo

Inspired by my family, who like to make hand-written to-do and shopping lists, but when it comes time to use them to share with each other it makes it difficult to keep track of who is in charge of each task. So I wanted to make an easy to use list maker that also allows you to share the list with someone else and can see when each other accomplishes a certain task.

This app allows users to create their own account and create lists that they then have the option to share with others.

Technology Client: React, ReactRouter, HTML, CSS

Techonology API: Node.js, Express, PostgreSQL

Best Seller Book Explorer

Live App Client Repo

I love to read, but often times I have a hard time figuring out what I would like to read. So I wanted to create an application that gave users a different medium in the form of YouTube videos, that often include interviews with the author or book reviews, to help users decide on their next read.

Users can select which NYT best seller's list to look through and in return for each book on the list they will get a picture of the book cover, the name of the book, the author(s), a description, the option to display YouTube videos for the book, and a purchase link.

Technology Client: Javascript, jQuery, HTML, CSS

API's Used: New York Times API, YouTube API
